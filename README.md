# Assistia - AI Learning Assistant

Assistia is an interactive AI learning platform built around guided conversations.  
Users select a topic, explore concepts through a chatbot interface, take auto-graded quizzes, and track their progress over time.  
The goal of the project is to make AI education simple, conversational, and structured.

Built with **FastAPI**, **MongoDB**, and **Together AI**.

---

## Features

### User Authentication
- Secure registration and login  
- Session-based authentication  
- Password hashing using BCrypt  

### Conversational Learning
- Real-time message display  
- Typing animation for a natural feel  
- Topic-specific learning flows  
- AI-generated explanations, examples, and resources  

### Interactive Quizzes
- Auto-generated questions  
- Automatic grading  
- Progress tracking tied to each user  

### Topic System
- Structured topic-based learning  
- Subtopics on demand  
- Recommended paths and expansions  

### Chat History
- Full message retention  
- Resume previous sessions  
- Stored securely in MongoDB  

---

## Tech Stack

- **Backend:** FastAPI  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** MongoDB (Motor)  
- **AI Provider:** Together AI  
- **Templating:** Jinja2  
- **Auth & Security:** python-jose, Passlib, BCrypt  
- **Server:** Uvicorn
